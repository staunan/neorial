Start MongoDB

MongoDB is installed on this machine. Just use the following command to start MongoDB:

# MongoDB Start
sudo service mongod start

# MongoDB Stop
sudo service mongod stop

# MongoDB Restart
sudo service mongod restart

# MongoDB Use
mongo

Connect to MongoDB Server from Command Line

First you need to add MongoDB installation path to environment variable. Generally, the MongoDB server installation directory is C:\Program Files\MongoDB\Server\5.0\bin. Add this directory to your environment variable.

Open CMD and print the version. If it is successful, you will now be able to connect to your server:

mongo --version

You can use a connection string to connect:

mongo "mongodb://localhost:27017/database_name?readPreference=primary&directConnection=true&ssl=false"

If the connection is successful, you will enter in mongo shell, where you can now perform db query.

Backup your database

mongodump --db admin -o C:\backup

Restore your Database

mongorestore --db nft-marketplace C:\backup\admin
