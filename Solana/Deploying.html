Deploying

Requirements

  • NodeJS v14 or greater & NPM
  • The latest stable Rust build
  • Solana CLI v1.7.11 or later
  • Git

Clone the repo

We will use the following repo for our example:

https://github.com/solana-labs/example-helloworld

After cloning the repo, run npm install to install packages.

Build the on-chain program

There is both a Rust and C version of the on-chain program. Open CMD in administrative mode and enter one of the following command:

npm run build:program-rust
npm run build:program-c

Deploy the on-chain program

solana program deploy dist/program/helloworld.so

Run the JavaScript client

npm run start
