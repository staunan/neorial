Array

Creating Array

			dataType[] arrayRefVar;   // preferred way.
			or
			dataType arrayRefVar[];  // works but not preferred way.
All the elements inside the array must be similar type of data. We cannot have mixed type of data like, you cannot store integer and float value together within the same array.

Example -

			double[] myList;   // preferred way.
			or
			double myList[];   // works but not preferred way.

You can also create an array using new operator.

			arrayRefVar = new dataType[arraySize];

The above statement does two things −

  • It creates an array using new dataType[arraySize].
  • It assigns the reference of the newly created array to the variable arrayRefVar.

Declaring an array variable, creating an array, and assigning the reference of the array to the variable can be combined in one statement, as shown below −

			dataType[] arrayRefVar = new dataType[arraySize];

Alternatively you can create arrays as follows −

			dataType[] arrayRefVar = {value0, value1, ..., valuek};

Accessing Array

The array elements are accessed through the index. Array indices are 0-based; that is, they start from 0 to arrayRefVar.length-1.

			dataType[] arrayRefVar = {value0, value1, ..., valuek};
			var theValue = arrayRefVar[2]; 
			//Accessing the third element. As the first element is indexed as 0

Example -

		public class TestArray {

		   	public static void main(String[] args) {
		      double[] myList = {1.9, 2.9, 3.4, 3.5};

		      // Print all the array elements
		      for (int i = 0; i < myList.length; i++) {
		         System.out.println(myList[i] + " ");
		      }
		     
		      // Summing all elements
		      double total = 0;
		      for (int i = 0; i < myList.length; i++) {
		         total += myList[i];
		      }
		      System.out.println("Total is " + total);
		      
		      // Finding the largest element
		      double max = myList[0];
		      for (int i = 1; i < myList.length; i++) {
		         if (myList[i] > max) max = myList[i];
		      }
		      System.out.println("Max is " + max);  
		   }
		}

Output -

			1.9
			2.9
			3.4
			3.5
			Total is 11.7
			Max is 3.5

Enhanced For Loop

JDK 1.5 introduced a new for loop known as foreach loop or enhanced for loop, which enables you to traverse the complete array sequentially without using an index variable.

			public class TestArray {
			   	public static void main(String[] args) {
					double[] myList = {1.9, 2.9, 3.4, 3.5};
					// Print all the array elements
					for (double element: myList) {
						System.out.println(element);
					}
				}
			}

Here, in the first element of the array gets assigned to the variable element. And in the second iteration, the second element is get automatically assigned to the variable element and so on until, the last element of the array comes. When there's no more element, the loops gets terminated automatically. Note the datatype of the element must be the same as the datatype of the element of the array.
