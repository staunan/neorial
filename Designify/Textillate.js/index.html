How To Use Textillate.js

Dependencies

jQuery, lettering.js and animate.css

Mark Up

<h1 class="tlt">My Title</h1>

Initialization

When the DOM is ready - 

$('.tlt').textillate();

This will initialize with the default options -

Attributes

data-in-effect - This will take the value as the keyframe name for in animation

data-out-effect - This will take the value as the keyframe name for out animation

data-in-shuffle - Boolean. True value specifies that the animation will happen on shuffling order of the letters for in animation. Non in sequence.

data-out-shuffle - Boolean. True value specifies that the animation will happen on shuffling order of the letters for out animation. Not in sequence.

Options

in

Instead of using data-in-effect attribute we can also specify the keyframe using option -

// in animation settings
$('.tlt').textillate({ 
	in: { effect: 'rollIn' } 
});

in: {
// set the effect name
effect: 'fadeInLeftBig',

// set the delay factor applied to each consecutive character
delayScale: 1.5,

// set the delay between each character
delay: 50,

// set to true to animate all the characters at the same time
sync: false,

// randomize the character sequence
// (note that shuffle doesn't make sense with sync = true)
shuffle: false,

// reverse the character sequence
// (note that reverse doesn't make sense with sync = true)
reverse: false,

// callback that executes once the animation has finished
callback: function () {}
},

out

// out animation settings.
out: {
	effect: 'hinge',
	delayScale: 1.5,
	delay: 50,
	sync: false,
	shuffle: false,
	reverse: false,
	callback: function () {}
},

selector

The default selector to use when detecting multiple texts to animate

selector: '.texts'

minDisplayTime

sets the minimum display time for each text before it is replaced. Delay between in and out animation.

minDisplayTime: 2000

loop

Boolean. This will enable looping when it's true. It means the animation will start over after it has finished. And it will continue this way.

loop:true

autoStart

Boolean. When it sets to true, the animatio will start automatically on page load.

autoStart : true

callback

Callback that executes once textillate has finished
callback: function () {},

type

Set the type of token to animate (available types: 'char' and 'word')

Events

Textillate triggers the following events:
  • start.tlt - triggered when textillate starts
  • inAnimationBegin.tlt - triggered when the in animation begins
  • inAnimationEnd.tlt - triggered when the in animation ends
  • outAnimationBegin.tlt - triggered when the out animation begins
  • outAnimationEnd.tlt - triggered when the in animation ends
  • end.tlt - triggered when textillate ends
	$('.tlt').on('inAnimationBegin.tlt', function () {
	  // do something
	});

Methods

  • $element.textillate('start') - Manually start/restart textillate
  • $element.textillate('stop') - Manually pause/stop textillate
  • $element.textillate('in') - Trigger the current text's in animation
  • $element.textillate('out') - Trigger the current text's out animation
