Imaginate - Image Hover Styles - How To Use

First include Vue.js, jQuery, animate.css, imaginate.css and imaginate.js.

Then copy the following code - 

		<image-overlay image-url = "Picture.jpg" in = "zoomIn" out = "zoomOut" id = "anything">
			<div>
				<h1>Anything</h1>
			</div>
		</image-overlay>

In - This property takes the animation name that will happen when you mouse over the image.

Out - This property takes the animation name that will happen when you take out the mouse out of the image.

Animation names are the keyframe from animate.css. Any animation keyframe is applicable.
