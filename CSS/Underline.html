Underline

text-decoration-line

The text-decoration-line property is used to set or remove decorations from text. The value text-decoration-line: none; is often used to remove underlines from links:

a {
    text-decoration-line: none;
}

You can also use other values:

// Creates a line above the text.
h1 {
    text-decoration-line: overline;
}

// Just like strike-through text. Line over the text.
h2 {
    text-decoration-line: line-through;
}

// line below the text.
h3 {
    text-decoration-line: underline;
}

text-decoration-style

The text-decoration-style property sets the style of the text decoration (like solid, wavy, dotted, dashed, double).

div.a {
    text-decoration-line: underline;
    text-decoration-style: solid;
}

div.b {
    text-decoration-line: underline;
    text-decoration-style: wavy;
}

div.c {
    text-decoration-line: underline;
    text-decoration-style: double;
}

div.d {
    text-decoration-line: overline underline;
    text-decoration-style: wavy;
}

<div class="a">Solid</div>
<div class="b">Wavy</div>
<div class="c">Double</div>
<div class="d">Wavy overline and underline</div>

The above code will output the following:

Solid
Wavy
Double
Wavy overline and underline

text-decoration-color

The text-decoration-color property specifies the color of the text-decoration (underlines, overlines, linethroughs).

p.colored_underline {
    text-decoration: underline;
    text-decoration-color: red;
}
<p class="colored_underline">Hello World</p>

Hello World

text-decoration-thickness

The text-decoration-thickness property specifies the thickness of the decoration line.

p.thickness_underline {
    text-decoration-line: overline;
    text-decoration-thickness: 20px;
    text-decoration-color: blue;
    text-decoration-style: dashed;
}
<p class="thickness_underline">Hello World</p>

text-underline-position

The text-underline-position CSS property specifies the position of the underline which is set using the text-decoration property's underline value.

p.underline_position_under {
    text-decoration-line: underline;
    text-underline-position: under;
}
p.underline_position_left {
    text-decoration-line: underline;
    text-underline-position: left;
}
p.underline_position_right {
    text-decoration-line: underline;
    text-underline-position: right;
}
p.underline_position_auto {
    text-decoration-line: underline;
    text-underline-position: auto;
}
<p class="underline_position_under">Underline Position : Under</p>
<p class="underline_position_left">Underline Position : Left</p>
<p class="underline_position_right">Underline Position : Right</p>
<p class="underline_position_auto">Underline Position : Auto</p>

Underline Position : Under

Underline Position : Left

Underline Position : Right

Underline Position : Auto

Here are the values you can use with this property:

  1. under: Forces the line to be set below the alphabetic baseline, at a position where it won't cross any descenders. This is useful for ensuring legibility with chemical and mathematical formulas, which make a large use of subscripts.
  2. left: In vertical writing-modes, this keyword forces the line to be placed on the left side of the text. In horizontal writing-modes, it is a synonym of auto.
  3. right: In vertical writing-modes, this keyword forces the line to be placed on the right side of the text. In horizontal writing-modes, it is a synonym of auto.

text-underline-offset

The text-underline-offset property in CSS sets the distance of text underlines from their initial position. Once you apply an underline for an element using text-decoration with the value of underline, you can say how far that line should be from your text using the text-underline-offset property.

p.underline_position_offset {
    text-decoration-line: underline;
    text-underline-offset: 20px;
}

<p class="underline_position_offset">Underline Offset : 20px</p>

Underline Offset : 20px

Note that, it doesn't work on other text-decoration lines, such as line-through or overline. You can also use negative value to make the underline appear on top direction.

text-decoration

The text-decoration property specifies the decoration added to text, and is a shorthand property for:

  1. text-decoration-line (required)
  2. text-decoration-color
  3. text-decoration-style
  4. text-decoration-thickness
text-decoration: text-decoration-line text-decoration-color text-decoration-style text-decoration-thickness;

For example,

h1 {
    text-decoration: underline overline dotted red;
}

h2 {
    text-decoration: underline wavy blue 5px;
}
