Links

With CSS we can customize anchor tag in different ways. A link has different state. You can display style to indicate not visited link or visited link. You can also display different style when the link is hovered. Or different style when link is active. You can use the following pseudo class to specify different css rules depending on different state:

  1. a:link: A normal, unvisited link.
  2. a:visited: A link the user has visited.
  3. a:hover: A link when the user puts his mouse over it.
  4. a:active: A link the moment it is clicked.

The above pseudo class is described in Chapter Pseudo Class.

Order of pseudo class

When setting the style for several link states, there are some order rules:

  1. a:hover MUST come after a:link and a:visited.
  2. a:active MUST come after a:hover.

Removing Default style from Anchor tag

By default browser render links with an underline below the text. If you want to remove that underline use text-decoration property with the value none.

.remove_link{
    text-decoration: none;
}
