Address Format

There are two type of address you can use in Zilliqa, and Base16.

  • Bech32 : This type of address starts with zil. For example, zil1l3evdmlqla0ca02vj3jjy3npa9n0p7fz730aum
  • ByStr20 : This type of address starts with 0x. These are hexadecimal format. For example, 0xfc72c6Efe0FF5f8ebD4C9465224661E966F0f922.

Your ZillPay wallet displays the address in Beach32 format by default. To change the format you can go to Settings->Advanced->Address Format, and select Base16.
