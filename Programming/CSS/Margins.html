Margin Left

The margin-left adds margin on the left side of the element. It creates an space on the left side of the element.

You can apply any unit value to the margin like em, rem, percentage, px etc. If you use percentage, the percentage value is relative to it's parent element's width and height. If your parent element is of 100px width, then setting margin on left or right side or both to it's child element to 20% results 20px margin. Same rules for the height too. 

			div{
				margin-left : 100px;
			}

Margin Right

The margin-right adds margin on the right side of the element. It creates an space on the right side of the element. 

			div{
				margin-right : 100px;
			}

Margin Top

The margin-top adds margin on the top side of the element. It creates an space on the top side of the element. 

			div{
				margin-top : 100px;
			}

Margin Bottom

The margin-bottom adds margin on the bottom side of the element. It creates an space on the bottom side of the element. 

			div{
				margin-bottom : 100px;
			}

Margin

The margin specifies the margin on all four side of the element. It is shorthand syntax of all the above four property. 

			div{
				// Add margin on all four side of the element.
				margin : 10px;

				// Top and bottom is 10px and left and right is 20px
				margin : 10px 20px;

				// Top is 10px, left and right side is 20px and bottom is 30px;
				margin : 10px 20px 30px;

				// Starting from the Top and it goes clockwise - top, right, bottom and left.
				margin : 10px 20px 30px 40px;
			}

Negative Value

The auto Keyword

You can set the margin property to auto to horizontally center the element within its container. The element will then take up the specified width, and the remaining space will be split equally between the left and right margins: 

			div {
			    width: 300px;
			    margin: auto;
			    border: 1px solid red;
			}
