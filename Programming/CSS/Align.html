Center Align Elements

To horizontally center a block element (like <div>), use margin: auto; Setting the width of the element will prevent it from stretching out to the edges of its container. The element will then take up the specified width, and the remaining space will be split equally between the two margins. You still can specify margin-top and margin-bottom.

Note: Center aligning has no effect if the width property is not set (or set to 100%).

In the following demo, the red box is a square of 100px, but using margin: auto;, the box is positioned centered.

<style>
	.centerAlignBlockElement {
		margin: auto;
		width: 100px;
		height: 100px;
		border: 3px solid green;
		padding: 10px;
		background: red;
	}
</style>
<div class="centerAlignBlockElement"></div>

Center Align Text

To just center the text inside an element, use text-align: center;.

<style type="text/css">
	.centerAlignText {
		text-align: center;
		border: 3px solid green;
		width: 600px;
		padding: 10px;
	}
</style>

<div class="centerAlignText">Hello World</div>
Hello World

Center Vertically - Using padding

There are many ways to center an element vertically in CSS. A simple solution is to use top and bottom padding:

<style type="text/css">
	.paddingMiddle {
		padding: 70px 0;
		border: 3px solid green;
		width: 100%;
	}
</style>

<div class="paddingMiddle">
	Only I can change my life. No one can do it for me.
</div>
Only I can change my life. No one can do it for me.

Note that, in the above example, the text looks exactly in the middle. It is possible as there is no height property set on the element. The moment you set height property, you won't be able to center the text most of the time.

	<style type="text/css">
		.paddingMiddle {
			padding: 70px 0;
			border: 3px solid green;
			width: 100%;
			height: 500px;
		}
	</style>
	
	<div class="paddingMiddle">
		Only I can change my life. No one can do it for me.
	</div>
Only I can change my life. No one can do it for me.

Center Vertically - Using line-height

Another trick is to use the line-height property with a value that is equal to the height property.

<style>
	.centerLineHeight{
		border: 2px solid green; 
		height: 400px;
		line-height: 400px;
		width: 100%;
	}
</style>
<div class="centerLineHeight">
	Optimism is the faith that leads to achievement. Nothing can be done without hope and confidence.
</div>
Optimism is the faith that leads to achievement. Nothing can be done without hope and confidence.

Center Vertically - Using position & transform

If padding and line-height are not options, a third solution is to use position and the transform property:

<style type="text/css">
	.usingTransform{
		border: 2px solid green;
		height: 200px;
		width: 800px;
		position: relative;
	}
	.usingTransform p{
		margin: 0;
		position: absolute;
		top: 50%;
		left: 50%;
		transform: translate(-50%, -50%);
		background-color: pink;
	}
</style>

<div class="usingTransform">
	<p>With the new day comes, new strength and new thoughts.</p>
</div>

With the new day comes, new strength and new thoughts.

In the above example, the p element is perfectly centered positioned. The css rule position: absolute; top: 50%; left: 50%; makes the paragraph start from center. To make it excatly in the middle, we need to shift the paragraph in opposite direction by half. That way, the center coordinate of paragraph, and center coordinate of outer element overlaps each other.

vertical-align

The vertical-align property in CSS controls how elements set next to each other on a line are lined up.

img {
  vertical-align: middle;
}

In order for this to work, the elements need to be set along a baseline. As in, inline (e.g. <span>, <img>) or inline-block (e.g. as set by the display property) elements.

<style type="text/css">
	.middle, .baseline, .top, .bottom{
		border: 2px solid green;
		box-sizing: border-box;
		font-size: 0;
	}
	.one{
		height: 50px;
		width: 50px;
		background: red;
		display: inline-block;
		font-size: 15px;
	}
	.two{
		height: 100px;
		width: 100px;
		background: yellow;
		display: inline-block;
		font-size: 15px;
	}
	.three{
		height: 50px;
		width: 50px;
		background: green;
		display: inline-block;
		font-size: 15px;
	}
	.middle div{
		vertical-align: middle;
	}
	.baseline div{
		vertical-align: baseline;
	}
	.top div{
		vertical-align: top;
	}
	.bottom div{
		vertical-align: bottom;
	}
</style>
<div>
	<h3>Middle</h3>
	<div class="middle">
		<div class="one"></div>
		<div class="two"></div>
		<div class="three"></div>
	</div>
	<h3>Baseline</h3>
	<div class="baseline">
		<div class="one"></div>
		<div class="two"></div>
		<div class="three"></div>
	</div>
	<h3>Top</h3>
	<div class="top">
		<div class="one"></div>
		<div class="two"></div>
		<div class="three"></div>
	</div>
	<h3>Bottom</h3>
	<div class="bottom">
		<div class="one"></div>
		<div class="two"></div>
		<div class="three"></div>
	</div>
</div>

Other values that you can use are:

  1. text-top : Aligns the top of the element with the top of the parent element's font.
  2. text-bottom : Aligns the bottom of the element with the bottom of the parent element's font.
  3. sub : Aligns the baseline of the element with the subscript-baseline of its parent. Like where a <sub> would sit.
  4. sup : Aligns the baseline of the element with the superscript-baseline of its parent. Like where a <sup> would sit.
  5. length : Aligns the baseline of the element at the given length above the baseline of its parent. (e.g. px, %, em, rem, etc.)

These values are very insignificant as they are used very rarely.
