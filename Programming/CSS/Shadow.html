Shadow

In CSS, there are two types of shadow:

  1. text-shadow
  2. box-shadow

text-shadow

Text shadow is used to draw shadow on text.

// Single Shadow
p { 
  text-shadow: 1px 1px 1px #000;
}

// Multiple Shadow
p { 
  text-shadow: 1px 1px 1px #000, 3px 3px 5px blue; 
}

The first two values specify the length of the shadow offset. The first value specifies the horizontal distance and the second specifies the vertical distance of the shadow. The third value specifies the blur radius and the last value describes the color of the shadow.

Using positive numbers as the first two values ends up with placing the shadow to the right of the text horizontally (first value) and placing the shadow below the text vertically (second value).

Using negative numbers as the first two values ends up with placing the shadow to the left of the text horizontally (first value) and placing the shadow on the top of the text vertically (second value).

blur value is optional

The third value, the blur radius, is an optional value which can be specified but doesn't have to. It's the amount of pixels the text is stretched which causes a blur effect. If you don't use the third value it is treated as if you specified a blur radius of zero.

Box Shadow

The property box-shadow is used to apply shadow on a box.

.shadow{
	box-shadow: 3px 3px 5px 6px #ccc;
}

box-shadow: [horizontal offset] [vertical offset] [blur radius] [optional spread radius] [color];

Following is the explaination of each part of the property box-shadow:

  1. The horizontal offset: [Required] The positive value means the shadow will be on the right of the box, a negative offset will put the shadow on the left of the box.
  2. The vertical offset: [Required] The negative value means the shadow will be above the box, a positive value position the shadow below the box.
  3. The blur radius: [Required] If the value set to 0, the shadow will be sharp, the higher the number, the more blurred it will be, and the further out the shadow will extend. For instance a shadow with 5px of horizontal offset that also has a 5px blur radius will be 10px of total shadow.
  4. The spread radius: [Optional] The positive value increases the size of the shadow, negative value decreases the size. Default is 0 (the shadow is same size as blur).
  5. Color: [Required] It takes any color value, like hex, named, rgba or hsla. If the color value is omitted, box shadows are drawn in the foreground color (text color). But be aware, older WebKit browsers (pre Chrome 20 and Safari 6) ignore the rule when color is omitted.

Inner Shadow

Inner shadow is same as the outer shadow. Instead it draws the shadow inside the border. Using inset keyword before the box-shadow value creates inner shadow. Here is an example:

box-shadow: inset 0 0 10px #000000;

One Side Shadow

Using a negative spread radius, you can get squeeze in a box shadow and only push it off one edge of a box.

box-shadow: 0 8px 6px -6px black;
