iziToast

This is one hell of a toasting framework. It's so tasty and yummy. Include the following files - 

			<script type="text/javascript" src="Assets/iziToast/iziToast.min.js"></script>
    		<link rel="stylesheet" type="text/css" href="Assets/iziToast/iziToast.min.css">

Basic Toast

iziToast.show({ title: 'Hey', message: 'What would you like to add?' }); iziToast.info({ title: 'Hello', message: 'Welcome!', }); iziToast.success({ title: 'OK', message: 'Successfully inserted record!', }); iziToast.warning({ title: 'Caution', message: 'You forgot important data', }); iziToast.error({ title: 'Error', message: 'Illegal operation', });

Options

In addition to title and message it provides a lots of options that you can use -

class The class that will be applied to the toast. It may be used as a reference.
titleColor Title color.
titleSize Title font size.
titleLineHeight CSS lineheight of the title.
messageColor the color of the message.
messageSize The font size of the message.
messageLineHeight Message Line Height
backgroundColor The background color of the toast.
theme light or dark
color Theme color of the Toast.
icon For font-awesome, use it like this fa fa-envelop.
iconText If you are using Material Icon, you can use the icon name. like user
iconColor Icon color.
image The value is the location of the image.
maxWidth The maximum width of the toast.
zindex zindex of the Toast.
layout 1 or 2. 1 is default. If you set it to 2, the toast window will enlarge.
balloon Default is false. Applies a balloon like toast.
close Set it to true, if you want the close button.
closeOnEscape Allows to close toast using the Esc key.
position Where it will be shown. It can be bottomRight, bottomLeft, topRight, topLeft, topCenter, bottomCenter or center.
timeout Amount in milliseconds to close the toast or false to disable.
pauseOnHover Pause the toast timeout while the cursor is on it.
overlay Enables display the Overlay layer on the page.
overlayClose Allows to close toast clicking on the Overlay.
overlayColor Overlay background color.
transitionIn Default toast open animation. It can be: bounceInLeft, bounceInRight, bounceInUp, bounceInDown, fadeIn, fadeInDown, fadeInUp, fadeInLeft, fadeInRight or flipInX.
transitionOut Default toast close animation. It can be: fadeOut, fadeOutUp, fadeOutDown, fadeOutLeft, fadeOutRight, flipOutX
progressBarColor The color of the Progress bar.
