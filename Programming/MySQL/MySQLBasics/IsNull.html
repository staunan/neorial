IS NULL Operator

To test whether a value is NULL or not, you use the IS NULL operator. Here's the basic syntax of the IS NULL operator:

value IS NULL

If the value is NULL, the expression returns true. Otherwise, it returns false.

Because the IS NULL is a comparison operator, you can use it anywhere that an operator can be used e.g., in the SELECT or WHERE clause. See the following example:

SELECT 1 IS NULL,  -- 0
    0 IS NULL,  -- 0
    NULL IS NULL; -- 1

IS NOT NULL

To check if a value is not NULL, you use IS NOT NULL operator:

value IS NOT NULL

This expression returns true (1) if the value is not NULL. Otherwise, it returns false (0). Consider the following example:

SELECT 1 IS NOT NULL, -- 1
    0 IS NOT NULL, -- 1
    NULL IS NOT NULL; -- 0

Example 1: Sample Database

The following query uses the IS NULL operator to find customers who do not have a sales representative:

SELECT 
    customerName, 
    country, 
    salesrepemployeenumber
FROM
    customers
WHERE
    salesrepemployeenumber IS NULL
ORDER BY 
    customerName;
