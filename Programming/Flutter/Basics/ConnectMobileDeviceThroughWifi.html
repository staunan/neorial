Connect your Mobile Device through Wi-Fi

You can also test your application on your mobile device through Wifi. There is a program called adb through which you can connect your device. To know more info about adb go to the following link. ADB.

The ADB tool is installed by default when you install Android Studio. ADB is included in the Android SDK Platform-Tools package. You can download this package with the SDK Manager, which installs it at android_sdk/platform-tools/.

Here are the steps you need to follow:

  • First enable the Wireless debugging in Developer Options on your mobile device:

  • First make sure that your mobile device and your computer is connected to the same router.
  • Copy the IP address of your mobile device. To obtain the IP address go to Settings > About Phone > Status Information > IP Address
  • Connect your mobile device through USB.
  • Test your device by pinging it. From terminal type ping mobile_ip_address. Go furthur if your computer successfully able to ping your mobile device.
  • In terminal cd into your ADB path, the path generally would look like C:\Users\NeoRe\AppData\Local\Android\Sdk\platform-tools
  • Run the command : adb kill-server
  • Run the command : adb usb
  • Run the command : adb tcpip 5555
  • Unplug USB cable.
  • Run the command : adb connect mobile_ip_address
  • Run the command : adb devices. This will list your mobile device.

Now you can run the command flutter devices to see all the devices available. It should show your phone IP address in the device list.

Now everytime before you run the flutter project, you need to connect your devices through ADB. But only the following steps needed and you don't need to plug your USB cable anymore.

cd /
cd C:\Users\NeoRe\AppData\Local\Android\Sdk\platform-tools
adb tcpip 5555
adb connect mobile_ip_address
adb devices
