mj-button

Displays a customizable button.

<mjml>
  <mj-body>
    <mj-section>
      <mj-column>
        <mj-button font-family="Helvetica" background-color="#f45e43" color="white">
          Don't click me!
         </mj-button>
      </mj-column>
    </mj-section>
  </mj-body>
</mjml>

Here is the list of attribute you can use on button element -

  • background-color of the button.
  • container-background-color background-color of the container.
  • border
  • border-left
  • border-right
  • border-top
  • border-bottom
  • border-radius
  • font-style
  • font-size
  • font-family
  • font-weight
  • color
  • text-align
  • text-decoration
  • text-transform
  • align align the button within it's container. Values are left, right, center
  • vertical-align
  • line-height line height of the text.
  • href
  • rel
  • target
  • inner-padding padding of the button itself.
  • padding padding of the container.
  • padding-top padding top of the container.
  • padding-bottom padding bottom of the container.
  • padding-right padding right of the container.
  • padding-left padding left of the container.
  • width width of the button.
  • height height of the button.
  • css-class

This element is rendered as a anchor tag. So you can apply href, rel, target attribute as you do with a tag in HTML.

Border properties are for the button itself.
