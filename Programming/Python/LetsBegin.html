Downloading and installing

Download Python software for Windows and install It. It will install two software, one is 64-bit Command promt, and another is IDLE which you can use to test python script. Once the installation is successful, open IDLE, to interact with the programming language.

Interactive Mode

Interactive Mode

In Interactive Mode, you can type any statement, and when you hit "Enter" key, the shell will print the output right away right at the bottom of your statement. This mode is called Interactive Mode. To try python in Interactive Mode, you can open IDLE which is installed when you installed Python.

Primary Prompt

In this interactive mode, the very first output you will see :

Python 3.6.3 (v3.6.3:2c5fed8, Oct  3 2017, 18:11:49) [MSC v.1900 64 bit (AMD64)] on win32
Type "copyright", "credits" or "license()" for more information.
>>>

In the above, the symbol >>> is called Primary Prompt. When you type, the typing will start after that prompt. That prompt takes whatever you type and executes it when you hit "Enter".

Secondary Prompt

The secondary prompt is ... (three dots). In this prompt, you can provide multiline statements. It is a line continuation prompt, allows you to type more than one line. And execute everything at onec.

Hello World

Let's print a Hello World message.

>>> print ("Hello World");

// Output :
Hello World

Number

Let's play around with numbers in that interactive mode.

Basic Arithmetic Operators

Basic four arithmetic operators are +, -, / and *. They perform Addition, Subtraction, Multiplication, and Division respectively. Here is a demo:

>>> 2 + 2
4
>>> 50 - 5*6
20
>>> (50 - 5*6) / 4
5.0
>>> 8 / 5  # division always returns a floating point number
1.6

The values, 2, 4, 5 etc have int type. And 1.6, 5.0 have float type.

Division always returns float value.

Floor Division

To do a floor division, you can use basic arithmetic operators and they will be bound to the last numeric digit. In case, the digit is negative, the fractional part will be type of number of frequency that the device generates.

In this architecture, the floor division is necesarry to print all the numbers from 0 and upwards.
