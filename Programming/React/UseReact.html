Use React

In this tutorial I'm going to show you how to use React.js in different way.

CDN

Include the following script just after the opening of body element in your HTML file.

<script src="https://unpkg.com/react@16/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@16/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/babel-standalone@6.15.0/babel.min.js"></script>

Now paste the following code in your HTML page and refresh the page, the component will be mounted on the page and you will see the output -

<div id="root"></div>

<script type="text/babel">
class Clock extends React.Component {
  constructor(props) {
    super(props);
    this.state = {date: new Date()};
  }

  render() {
    return (
      <div>
        <h1>Hello, world!</h1>
        <h2>It is {this.state.date.toLocaleTimeString()}.</h2>
      </div>
    );
  }
}

ReactDOM.render(
  <Clock />,
  document.getElementById('root')
);
</script>

In this approach we are using In-Browser Babel transformer. This approach is useful only for demos and very small react application. Because each time the page loads, Babel takes some time to transform the React code to ES5 script.

CDN - Without Babel Transformer

Before we were using Babel transformer to transform JSX Code to pure JS code. However, if you don't want to use JSX, you need to pull in only following two file. Copy and paste the following just after body element in your HTML page - 

<script src="https://unpkg.com/react@16/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@16/umd/react-dom.development.js"></script>

Now paste the following code and refresh the page -

<script type="text/babel">
const e = React.createElement;

class LikeButton extends React.Component {
  constructor(props) {
    super(props);
    this.state = { liked: false };
  }

  render() {
    if (this.state.liked) {
      return 'You liked this.';
    }

    return e(
      'button',
      { onClick: () => this.setState({ liked: true }) },
      'Like'
    );
  }
}

const domContainer = document.querySelector('#like_button_container');
ReactDOM.render(e(LikeButton), domContainer);
</script>

Using JSX Preprocessor

In this way, you will transform your JSX to ES5 using Command. The benifit of this method is that your code will already be compiled to ES5 script. So there will be no transformation latency in the Browser. Also you can watch a whole folder and compile many js file together and whenever you make any changes to any file, the preprocessor will compile the file automatically and generate the compiled version instantly. This way your development time will speed up. If you have Node.js installed you can follow the following steps -

  • Create a new folder. Let's say the folder name is Test
  • Open CMD and change your current directory to Test folder.
  • run npm init -y. This will create a package.json file with some content.
  • Next, run the commands - npm install babel-cli@6 babel-preset-react-app@3. This will create a folder node_modules which will contains all the installation files.
  • Now create a folder js inside Test folder. Open js folder.
  • Create two folder inside js - src and build
  • After successfully installation of above command run npx babel --watch js/src --out-dir js/build --presets react-app/prod.

That's it. Now keep the Console open while you are developing react app. If you create a js file inside src folder, there will be a corresponding file created instantly and automatically with the same name in the build folder. In your HTML file you will refer to the file which are in the build folder. The files in the build folder will be the compiled file from src folder. The console will keep watching for the src folder for any changes. If you make any changes to any file in the src folder, the transformer will rerun automatically and update the build folder.

Create React App

Create React App is a comfortable environment for learning React, and is the best way to start building a new single-page application in React. It sets up your development environment so that you can use the latest JavaScript features, provides a nice developer experience, and optimizes your app for production.

Run the following command -

npx create-react-app my-app
cd my-app
npm start

The first command will take some time to install all it needs. After successfully installation and executing the command npm start, it will start a server automatically. Now open the browser and go to http://localhost:3000/, you will see the default page.

Now open up src/App.js, and make some changes, the build process will run automatically after you save the file and also you don't need to refresh the page on the browser, the updated build will be automatically pushed to the broswer. Awesome. This feature is called Hot Module Replacement(HMR) or Live-editing that comes from Webpack.
