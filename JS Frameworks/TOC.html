Ripple.js

This is a tiny framework that enables the ripple effect on your element. It's size is 4kb including it's CSS file.

ScrollAnim.js

A powerfull, smallest and efficient library to add animation to your element when you scroll up or down.

Intro.js

If you use facebook, you must have seen that they asked you to take a tour to know what's everything about in the page. It's kind of introduction to the user to make them familiar with the feature. With Vanilla Javascript it will take a lots of time. Intro.js provides a simple and clean API that makes this job easier for you.

SweetAlert.js

Different Browser renders alert differently. And Those are very basic native window without much styling. SweetAlert is the replacement. It's beautiful, elegent and easy to use.
